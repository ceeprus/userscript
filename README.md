# userscript

bunch of fast vibecoded stuff, not for your use

## Scripts

Click "Install" to open the raw `.user.js` file — if you have [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/) installed, it'll prompt you to install automatically.

| Script                                                               | Description                                                                                      | Install                                                                                                   |
| -------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------- |
| [YouTube: Hide Watched Videos](youtube/youtube-hide-watched.user.js) | Hides watched videos, Shorts, Mixes, and subscribed channels from your YouTube feeds.            | [Install](https://raw.githubusercontent.com/ceeprus/userscript/main/youtube/youtube-hide-watched.user.js) |
| [Backloggd - Copy Review Link](backloggd/copy-reviews.user.js)       | Adds an icon-only "copy link" button next to the "Open review" button on Backloggd review cards. | [Install](https://raw.githubusercontent.com/ceeprus/userscript/main/backloggd/copy-reviews.user.js)       |
| [FFXIV Language Redirect](ffxiv/ffxiv-redirect.user.js)              | Redirects any regional FFXIV site (DE/FR/EU/JP, etc.) to the NA version.                         | [Install](https://raw.githubusercontent.com/ceeprus/userscript/main/ffxiv/ffxiv-redirect.user.js)         |
| [GitHub Commit Email Revealer](github/user-mail.user.js)             | Shows all commit author emails in a popup next to "Browse Files", with a live status indicator.  | [Install](https://raw.githubusercontent.com/ceeprus/userscript/main/github/user-mail.user.js)             |
